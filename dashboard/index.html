Krypton NAS

From JSON spec to 3D-printed hardware

A 3D-printable Raspberry Pi NAS enclosure built entirely from a machine-readable spec. One JSON file defines every dimension, color, and constraint. A Python script generates parametric CAD geometry. A headless pipeline exports print-ready files. A vanilla JS dashboard visualizes it all. No frameworks, no build step.

End-to-End Pipeline

1 Spec Define dimensions, constraints, and interfaces in JSON
2 Generate FreeCAD Python script creates parametric 3D model
3 Export Pipeline assigns colors, exports multi-color 3MF + STLs
4 Visualize Three.js dashboard renders interactive 3D views
5 Print Slice, print, assemble with snap-fit mounts + magnets
6 Track State-driven JSON tracks components, features, tasks

About This Project

Built by a software engineer with zero CAD experience. Spec-driven development bridges the gap between software engineering and hardware design. Define everything in structured data, generate geometry from code, and track progress the same way you manage a software project.

Every dimension, color, and constraint lives in a JSON spec file. The FreeCAD Python script reads those parameters to generate 8 interlocking components. The pipeline exports print-ready files. And this dashboard renders it all live from the same state files. No manual sync, no stale documentation.
