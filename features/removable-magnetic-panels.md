# Feature: Removable Magnetic Panels

## Overview

4 side panels attach to the main shell via neodymium magnets, enabling tool-free access to all internal components.

## Status

- **Status**: completed
- **Version**: 1.0.0

## Design

Each panel has 6 magnet recesses aligned with matching recesses in the shell wall. Magnets press-fit into 1.5mm deep pockets (6mm diameter). The 5mm lip overlap provides alignment during attachment.

### Magnet Layout

- 6 magnets per side, 24 total
- Evenly distributed along panel edges
- Press-fit into both shell and panel recesses
- Polarity must be consistent across all panels
