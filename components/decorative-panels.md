# Component: Decorative Panels

## Overview

4 removable side panels with unique Krypton-themed patterns. Each attaches via 6 neodymium magnets for tool-free access.

## Common Specifications

| Property | Value |
|----------|-------|
| **Thickness** | 2.5 mm |
| **Lip overlap** | 5 mm |
| **Panel margin** | 15 mm from shell edges |
| **Magnets per panel** | 6 (6mm dia x 2mm) |
| **Print orientation** | Pattern face up, flat on bed |

## Front Panel - Crystal Burst

| Property | Value |
|----------|-------|
| **ID** | front-panel |
| **STL** | stl/FrontPanel_CrystalBurst.stl |
| **Pattern** | Radial diamond emanation |

Central diamond focal point with 3 expanding rings of crystal shapes radiating outward. Inspired by energy burst from a Kryptonian crystal.

## Rear Panel - Fortress Gate

| Property | Value |
|----------|-------|
| **ID** | rear-panel |
| **STL** | stl/RearPanel_FortressGate.stl |
| **Pattern** | Hex matrix + crystal slots |

Top crystal slot vents for exhaust airflow, hexagonal grid matrix in the center, and diamond accents at bottom. Includes RJ45 Ethernet passthrough cutout.

## Left Panel - Shield Array

| Property | Value |
|----------|-------|
| **ID** | left-panel |
| **STL** | stl/LeftPanel_ShieldArray.stl |
| **Pattern** | Shield tessellation |

Pentagon shield shapes inspired by the House of El crest. Includes functional cutouts for Raspberry Pi I/O:
- USB-C power port
- Dual micro-HDMI ports
- Ethernet jack

## Right Panel - Crystal Wave

| Property | Value |
|----------|-------|
| **ID** | right-panel |
| **STL** | stl/RightPanel_CrystalWave.stl |
| **Pattern** | Sinusoidal wave with diamonds |

Sinusoidal wave pattern with diamond nodes at peaks/troughs. Includes cutouts for:
- USB 3.0 passthrough
- Ethernet adapter access
- SSD cable exit slot

## Print Settings

| Setting | Recommended |
|---------|-------------|
| **Orientation** | Pattern face up |
| **Layer height** | 0.16mm (for pattern detail) |
| **Supports** | None needed |
| **Infill** | 100% (they're thin) |
| **Material** | PLA or PETG |
